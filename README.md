# Tabi-Tab

A simple list management web-app. 

## HOW TO RUN

### Download from Release:
1. Extract files
2. Open 'Tabi Tab.html'
3. Enjoy!

### From Source code:
>**Requirement: npm**.

1. install necessary dependencies for compiling:
```
"npm run get-dependencies"
``` 

2. build Tabi Tab:
``` 
"npm run compile"
``` 

3. Finally, run Tabi-Tab by open index.html in folder .\dist.
